# Homepage build-demo assets

The "Build in Seconds / With no Setup" section (components/BuildDemo.tsx) expects:

- `build-todo-app.mp4`   — the screen recording of building the todo app (16:9 recommended)
- `build-todo-app.jpg`   — a poster/still frame shown before play (same aspect ratio)
- `build-todo-app.vtt`   — (optional) captions; uncomment the <track> in BuildDemo.tsx

Keep the video compressed (H.264 MP4). Add a WebM <source> in BuildDemo.tsx for smaller files if desired.
